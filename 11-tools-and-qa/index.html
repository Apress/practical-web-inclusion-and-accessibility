			Today's lesson: Handy tools & apps

Because it's always a pain to type long web addresses from a book into a web browser, here are handy links for every Tool mentioned in Chapter 11 of the book.

The A11y Project

All-round useful website for all things accessibility - containing guides, how-to's, and an active blog covering changes in accessibility. Also the inventors of the term 'a11y'.

Lighthouse

Open-source tool for auditing web pages. Available inside Google Chrome as standard, and creates shareable reports that are scored out of 100.

  • Google Chrome Lighthouse overview

tota11y

A tool that provides a more visual way of debugging and auditing your site's accessibility. Added through a small piece of code, it can display content as a screen reader would read it on hover.

Google Chrome accessibility dev tools - overview

A panel inside the Google Chrome web inspector that can show the accessibility tree of a document, any active ARIA attributes, and computed properties.

Microsoft Edge accessibility dev tools - Overview

Available in the new Windows default browser. Capable of showing the accessibility tree of a page and ARIA attributes, as well as monitoring if an element has tabindex.

Firefox accessibility dev tools - Overview

The Firefox dev tools provide the same features as Google Chrome and Edge, but with a built-in contrast ratio checker for selected elements.

Gov UK - Designing for accessibility posters

A range of great starter posters highlighting simple do's and don'ts for a range of disabilities. Perfect for starting a discussion with companies about accessibility.

Gov UK - Persona profiles

A series of hypothetical users, with specific devices and access needs, that testers can use to take into account the experience for those with disabilities.

Gov UK - Informatiom about empathy labs

Information about what an 'empathy lab' is, and the one that the GDS set up for use throughout public sector to encourage the spread of accessibility awareness.

Paciello group - Screen reader cheat sheet

A cheat sheet of shortcuts for JAWS, Narrator, NVDA and VoiceOver screen readers for quick user reference.

Headings map browser extension

With one click, this extension displays a visual overview of the heading hierarchy of a page and any errors in its arrangement.

  • Google Chrome
  • Firefox

Validity browser extension

A quick in-browser HTML validator that displays any error or warnings in the browser's console.

  • Google Chrome
  • Firefox
  • Opera

Note: Microsoft Edge will soon support Google Chrome extensions.

WebAIM contrast checker

A website to allow for checking (and tweaking) of two colours to view their contrast levels. Also shows if they're AA and/or AAA WCAG compliant.

WebAIM link contrast checker

Another WebAIM site, this time to test whether the contrast level between a pair of colours works specifically for hyperlinks.

Contrast checker - Lea Verou

A simpler, more visual contrast checking site. This one allows you to quickly switch which colour is the background and which the foreground.

Chrome dev tools - Live contrast ratio checker

When clicking on an elements colour in 'Inspect Element', you can use the colour picker that displays AA and AAA contrast compliance levels.

Colorblind web page filter

Enter any URL and then choose a strain of colour blindness, and this site generates a side-by-side comparison of the site.

Sim Daltonism

Displays a window over your computer where you can switch between multiple strains of colour blindness to see what things look like.

WebVTT validator

A site that allows you to input the contents of your WebVTT files and then check them for syntax or content errors.

Converting a WebVTT file into transcript

A quick conversion of your WebVTT file will allow you to display it alongside any video or audio content to help users.

Ian Devlin - Programmaticly convert WebVTT files to transcripts

Achieve the same WebVTT conversion but through code - allowing you to handle the process automatically for any media.

Readableâ€™s online Flesch-Kincaid Reading Ease test

Input an article or your website copy to see how readable it is to users. Content is graded and the site offers improvements if possible.

How to report on mental health (for content creators)

A guide for those writing for TV, the news, or online about how to do so in a respectful way to those with mental health issues.

Design patterns for mental health

A joint project between the Public Policy Lab and web agency Snook to create a mental health-friendly pattern library.

Email CSS Support - Campaign Monitor

A site that breaks down the support for various areas of CSS in all popular email platforms.

HTML to TXT email converter - Mailchimp

A quick way to input the HTML for your email campaigns to generate a text equivalent you can send as a fallback/alternative.

Automated accessibility tools

AccessLint

A tool that automatically reviews code submitted to Github as a pull request and comments if it finds any issues. It also lets you know when you've fixed them.

Pa11y

Perfect for when you're developing. Pa11y can be run as a command or set up to watch files, and lets you know if what you've written is accessible or not.

A11y machine

Capable of generating easily understood accessibility reports for multiple URLs and sites at once. Ideal for comparing progress, or analysing the competition.

AATT

Created by Paypal, this tool takes in multiple accessibility checkers to create a comprehensive automated test-suite, available for any setup through an API.

WAVE evaluation tool

With a click of a button, provides a visual representation in a browser of a page's accessibility - what needs fixing and improving, but also any working features.

  • Google Chrome
  • Firefox

Auditing accessibility tools

Gov UK - Running research with those with disabilities

A guide written to help people planning to conduct their own research sessions with people who have disabilities. Includes advice for before, during, and after a session.

a11y project - list of professional help for accessibility

For if you feel that you need a helping hand when it comes to making your site(s) accessible. a11y project have collated some trusted accessibility consultants.
